
# Hospital Management API

Bu projemde temel bir hastane randevu sistemini yönetmek için geliştirilmiş  bir RESTful API uygulamasıdır. Python ve Flask kullanılarak geliştirilmiştir.

# Projemin Özellikleri :

 - Get ile randevuları listeleme 

 - Post ile yeni randevu oluşturma 

 - Put ile mevcut randevuları güncelleme

 - Delete ile randevuları silme 

 - Dinamik ID yönetimi ve hata kontrolleri.

# Projenin Kurulumu :
Projeyi yerel makinenizde çalıştırmak için;

Klasöre gidin:

'Bash
cd hospital-managiment-API
Gerekli kütüphaneyi kurun:

'Bash
pip install flask
Uygulamayı çalıştırın:

'Bash
python app.py
